## happymmall

`happymmall` 是一个基于 `SSM` 的企业级电商项目，非常适合作为新手的第一个 `Java` 线上项目实践。

## 项目模块

- 用户模块
- 分类模块
- 商品模块
- 购物车模块
- 收货地址模块
- 支付模块
- 订单模块

## 项目版本

项目主要使用 `Spring/SpringMVC/Mybatis/MySQL` 等技术，其中：
- `v1.0`：单服务器，主要使用 `Guava` 等；
- `v2.0`：`Tomcat` 集群 + `Redis` 分布式：主要使用 `Redis/Tomcat集群/Nginx` 等；
